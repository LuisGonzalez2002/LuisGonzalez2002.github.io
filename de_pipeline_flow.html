2 Etapas del pipeline Jenkins (35m 7s total)
01
PR Build — Preparation Steps
Instalación de dependencias, configuración del entorno dbt, validación de sintaxis YAML y SQL de los modelos modificados en la PR.
Jenkins dbt deps PR trigger
⚡ Auto
02
Feature → Release Tests
Ejecución de los modelos dbt en el schema de laboratorio de Snowflake. Lanzamiento de tests de calidad de datos (not_null, unique, relationships, custom tests).
dbt run dbt test LAB schema Jenkins
⚡ Auto
03
Release to Prod Tests
Validación completa del conjunto de modelos contra datos productivos en modo read-only. Verificación de la integridad referencial y volúmenes esperados.
dbt test --target prod PROD schema gate check
🔒 Gate
04
Commit to Lab Branch, Deploy Lab
Merge automático a rama lab. Deploy completo de los modelos al entorno de staging en Snowflake. Comprobación de materialización (tables, views, incrementals).
git merge lab dbt run --target lab LAB deploy
⚡ Auto
05
Merge to Main — Deploy to Production
Paso final: merge a main branch, deploy completo a producción en Snowflake. Generación de documentación dbt actualizada. Notificación de éxito. Duración: ~34m 50s.
merge → main dbt run --target prod PROD deploy dbt docs 34m 50s
✓ Deploy
